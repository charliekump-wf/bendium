# Bendium

Bendium is a Chrome extension for talking to Bender.

## Build

```
pub get
pub build extension
```

Install the `build/extension/` directory.

## Test

```
alias ddev='pub run dart_dev'
ddev gen-test-runner
ddev test -p content-shell
```
